# DropGoLine 抓擱來

**桌面應用程式開發 期末專題**

DropGoLine 是一款基於 P2P 技術的檔案傳輸與剪貼簿同步工具，專為區域網路內的高速傳輸設計。無需網際網路，即可在同一網段下的裝置間輕鬆分享檔案與文字。

## ✨ 功能特色

- **⚡ P2P 高速傳輸**: 直接在裝置間建立連線，傳輸速度僅受區域網路頻寬限制。
- **🖱️ 直覺拖放 (Drag & Drop)**: 支援將檔案直接拖入視窗即可傳送，操作就像移動本地檔案一樣簡單。
- **📋 剪貼簿同步**: 可設定自動同步裝置間的剪貼簿內容，在一台電腦複製，另一台電腦直接貼上。
- **📜 傳輸歷史**: 完整的檔案傳輸與文字訊息記錄，隨時回顧。
- **🔍 自動發現**: 自動掃描並顯示同一房間代碼內的線上用戶，免去手動輸入 IP 的麻煩。

| **Windows 10 x64** | 含 Runtime   | .zip     | 檔案較小，需先安裝 .NET Runtime。解壓縮後資料夾內執行。 |
| **Windows 10 x64** | 不含 Runtime | .exe     | 單一執行檔，檔案較大，隨點即用，無需安裝額外軟體。      |
| **Windows 11 x64** | 含 Runtime   | .zip     | 檔案較小，需先安裝 .NET Runtime。解壓縮後資料夾內執行。 |
| **Windows 11 x64** | 不含 Runtime | .exe     | 單一執行檔，檔案較大，隨點即用，無需安裝額外軟體。      |

## 使用教學

操作 demo 可以在 [youtube](https://youtu.be/vQg7fOnuj-U) 觀看

1.  **啟動程式**: 雙擊下載的執行檔 (或是解壓縮後的應用程式)。
2.  **設定房間代碼**: 在主畫面上方的房間代碼欄位輸入一個代碼 (代碼可以在在 notifyicon 右鍵選單獲取)，所有要互傳的裝置必須設定相同的代碼。
3.  **連接用戶**: 程式會自動搜尋同一房間內的用戶。點擊列表中的用戶頭像即可建立連線。
4.  **傳送檔案**:
    - A 機將檔案從檔案總管直接拖曳到目標用戶的卡片上。
    - B 機點擊用戶卡片，下載檔案後，檔案會自動保存到 A 機的預設下載目錄。隨後可以拖曳卡片到目標位置。
5.  **剪貼簿同步**:
    - 在設定中開啟「剪貼簿同步」功能。
    - 當您在一台裝置複製文字時，其他已連線且開啟同步功能的裝置將會自動更新剪貼簿。

## 開發與建置

如果您是開發者，可以按照以下步驟自行建置專案。

### 系統需求

- Windows 10 / 11
- .NET SDK 10.0 (或專案指定版本)

### 建置指令

使用 Visual Studio 開啟 `DropGoLine.sln` 或使用命令行工具：

```bash
# 發布單一執行檔 (不含 Runtime) 範例 - Windows x64
dotnet publish .\DropGoLine\ -r win-x64 --self-contained true -c Release -p:PublishSingleFile=true
```

如果你想一鍵發布所有版本，可以使用 `publish.ps1` 腳本。
